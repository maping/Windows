# Windows Terminal

## 1. 安装
在 Microsoft Store 搜索 windows terminal


## 2. Terminal 之间切换
- 从 Windows Terminal 切换到 Ubuntu Terminal: wsl
- 从 Windows Terminal 切换到 CMD Terminal: cmd

## 3. 窗口分屏，并创建快捷方式
- Alt + Shift + D, 持续分屏

shift+alt+"+" 横向分屏一个
shift+alt+"-" 竖向分屏一个

## 4. 删除文件时，如果提示 Permission Denyed，可能是因为有其它程序在使用该文件
