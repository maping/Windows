# Windows 版本
## 1. 查看 Windows 版本

按下快捷键：Win +R，然后输入 winver

## 2. Windows 版本说明

	1. Windows 10 Home - 家庭版，最基础的功能，适合普通用户。
	2. Windows 10 Pro - 专业版，在家庭版的基础上增加了设备管理和应用管理，保护企业敏感数据，支持远程和移动办公。
	3. Windows 10 Enterprise - 企业版，适合大客户使用。
	4. Windows 10 Education - 教育版，适合在校师生使用。
